## SQL - Exploratory Data Analysis (EDA)

### Overview
This project performs **Exploratory Data Analysis (EDA)** using SQL to understand **Database Exploration**, **Dimension Analysis**, **Date Analysis**, **Business Metrics**, **Magnitude Analysis** and **Ranking Analysis**  from a relational data warehouse.

## SQL - Sales Performance Analytics

### Overview
This project performs sales analytics using SQL to understand **business performance**, **customer behavior**, **product performance**, and **sales trends** from a relational data warehouse.
